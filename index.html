Data Handling and Privacy Policy

Effective Date: March 11, 2026

At LINSCO (official website: crocochic.com), we are committed to protecting the privacy and security of the data we process. This policy describes how we collect, use, store, share, and dispose of data obtained through the Amazon Selling Partner API (SP-API) for our application, SP-API Solicitations.

1. Data Collection

We collect Amazon Information solely through authorized Amazon SP-API calls. This information may include Order Personally Identifiable Information (PII) such as customer names, shipping addresses, and phone numbers, as well as non-PII data like order items, quantities, and pricing.

2. Data Processing and Usage

We process Amazon Information only for the specific purposes authorized by the selling partner, including order fulfillment, inventory management, and tax reporting. We do not use this data for marketing, profiling, or any other purposes not explicitly permitted by Amazon.

3. Data Storage and Security

All Amazon Information is stored securely. We enforce the following security controls:

  • Encryption at Rest: All Amazon Information is encrypted using industry-standard AES-256 encryption. We utilize a managed Key Management System (KMS) for secure key storage and rotation.
  • Credential Management: Access to systems handling Amazon data requires strong passwords (minimum 12 characters, including uppercase, lowercase, numbers, and symbols). We enforce a mandatory password expiration and rotation policy every 90 days. Multi-Factor Authentication (MFA) is required for all access.

4. Data Sharing and Disclosure

We do not share, sell, rent, or trade Amazon Personally Identifiable Information (PII) with third parties. Data is only accessed by our automated system processes necessary for the application's functionality.

5. Data Retention and Disposal

We retain Amazon Information only for as long as strictly necessary to fulfill the authorized purpose.

  • PII is purged from our active systems within 30 days of order fulfillment.
  • We perform permanent data disposal using cryptographic erasure to ensure data cannot be recovered.

6. Logging and Monitoring

We maintain a robust security logging and monitoring system:

  • Monitoring Mechanism: We use automated tools to monitor for suspicious activities, unauthorized access attempts, and API anomalies.
  • Log Retention: All security logs are retained for a minimum of 90 days.
  • PII Protection: We strictly exclude and do not store PII within our system logs.
  • Access Control: Access to logs is strictly restricted to authorized security administrators only.

7. Contact Us

If you have any questions about this Data Handling and Privacy Policy, please contact us at: contact@crocochic.com
